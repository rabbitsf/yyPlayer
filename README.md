This is a simple iOS MP3 Player app. 
Features and usage:
1. Tap the Add button to add a folder
2. Tap the Import button to import MP3 music files from the Files app. You will need to import your MP3 music files to the Files app first. To import MP3 files, tap on Import button, tap on Select or Create Folder. After selecting/creating a folder, tap on Import MP3 Files, navigate to where you upload your MP3 files in the Files app then select the music files.
3. Swipe left on a folder to delete a folder. Swipe left on a file to delete a file.
4. Song playing screen offers you to Play/Pause, Next and Prep, Repeat one song, Repeat All, and Shuffle play.
